GitHub Pages quick start
1) Create a new repo (e.g., waste-assistant)
2) Upload index.html from this folder to the repo root
3) Settings > Pages > Build and deployment: Source = Deploy from a branch; Branch = main, folder = / (root)
4) After save, your site will be published at https://<user>.github.io/waste-assistant/
5) In SharePoint, add the 'Embed' web part and paste the public URL of the site.
